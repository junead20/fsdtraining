# College Event Management System

## Overview

A modern, responsive web application for managing college events. Students can browse upcoming events, register for them, view detailed schedules, explore a gallery, and find answers to common questions.

## Pages

- **index.html** – Home page with hero section and highlights.
- **events.html** – Event cards with registration links.
- **registration.html** – Registration form with validation and success modal.
- **schedule.html** – Detailed multi‑day schedule table.
- **gallery.html** – Photo gallery with cards.
- **faq.html** – Accordion‑style FAQ.
- **contact.html** – Contact form.

All pages share a dark‑theme Bootstrap navbar, footer, and a back‑to‑top button.

## Tech Stack

- **HTML5**, **CSS3**, **JavaScript**
- **Bootstrap 5.3.3** for responsive components.
- **Bootstrap Icons** for icons.
- Custom CSS using CSS variables for a premium dark theme with gradients, glassmorphism, and smooth animations.

## Setup & Development

1. Clone or download the repository.
2. Open `index.html` in a browser (or serve the folder with any static server).
3. No build steps are required – all assets are static.
4. To modify styles, edit `css/style.css` and `css/custom-buttons.css`.
5. JavaScript logic lives in `js/script.js`.

## Features

- Responsive layout across devices.
- Navbar highlights the active page.
- Smooth scroll and back‑to‑top button.
- Form validation with user‑friendly alerts.
- Dynamic modals for registration success.
- Accessible with proper ARIA labels.
